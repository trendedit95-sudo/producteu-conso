Producteur-Consommateur
Simulation du problème classique Producteur-Consommateur en Java avec interface graphique Swing.
## Fonctionnalités
- Visualisation du tampon en temps réel
- Nombre de producteurs/consommateurs configurable
- Boutons Démarrer / Arrêter / Continuer / Réinitialiser
- Rapport final exportable

Lancer le projet
javac -d . src/Main/*.java
java -cp . Main.App

Technologies
- Java (Threads, Sémaphores)
- Swing (interface graphique)
